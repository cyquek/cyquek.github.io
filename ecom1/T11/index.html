Combining Logs and Interactions in a Regression

Q1

Construct a new logarithmic variable that involves the logarithm of age interacted with the female dummy variable:

  • female_log_age = female \(\times\) log(age)

Solution

Run the following code:

## Create logarthimic variables for ahe and age
mydata1$log_ahe=log(mydata1$ahe)
mydata1$log_age=log(mydata1$age)
# The as.numeric() command can be used to create dummy variables in R
## Create year of sample dummy variables for the regression
mydata1$d1992=as.numeric(mydata1$year==1992)  
#d1992 is a dummy variable that equals one if year=1992, and 0 otherwise
mydata1$d2012=as.numeric(mydata1$year==2012)  
#d2012 is a dummy variable that equals one if year=2012, and 0 otherwise

## Create interactive variables involving female and age
mydata1$female_age=mydata1$female*mydata1$age

## Create the interaction between female and log(age)
mydata1$female_log_age=mydata1$female*mydata1$log_age

Q2

Run the following 2 regressions, comment on the statistical significance of the coefficients involving age, female, and compute the following partial effects involving age or female:

  • Regression 1:
    • Dependent variable: log_ahe
    • Independent variables: age, bachelor, female, female_age, d1992
    • Compute the following partial effects:
      • Partial effect of being one year older if you are male
      • Partial effect of being one year older if you are female
      • Partial effect of being female if 25 years old
      • Partial effect of being female if 30 year old

\[\widehat{\log(ahe)} = \hat{\beta}_0 + \hat{\beta}_1 \, age+ \hat{\beta}_2 \, female \times age + \hat{\beta}_3 \, female + \hat{\beta}_4 \, bachelor + \hat{\beta}_5 \, d1992\]

  • Regression 2:
    • Dependent variable: log_ahe
    • Independent variables: log_age, bachelor, female, female_log_age, 1992
    • Compute the following partial effects (elasticities because of log-log form):
      • Partial effect / elasticity of being one year older if you are male
      • Partial effect / elasticity of being one year older if you are female

\[\widehat{\log(ahe)} = \hat{\beta}_0 + \hat{\beta}_1 \, \log(age)+ \hat{\beta}_2 \, female \times \log(age) + \hat{\beta}_3 \, female + \hat{\beta}_4 \, bachelor + \hat{\beta}_5 \, d1992\]

Solution

run the regressions

## Regression of log(ahe) on age, allowing for differential gender effects with bachelor, + controls
reg1=lm(log_ahe~age+female_age+female+bachelor+d1992,data=mydata1)
cov1=vcovHC(reg1, type = "HC1")    
se1=sqrt(diag(cov1))

## Regression of log(ahe) on log(age), allowing for differential gender effects with bachelor, + controls
reg2=lm(log_ahe~log_age+female_log_age+female+bachelor+d1992,data=mydata1)
cov2=vcovHC(reg2, type = "HC1")    
se2=sqrt(diag(cov2))

Q3

Using Regression 2, compute the elasticity of ahe with respect to age for females. Also report the 95% CI for this elasticity.

Solution

The predicted value from Regression 2 can be written as follows:

\[ \widehat{Y} = \widehat{log(age)} = \hat{\beta}_{0}+\hat{\beta}_{1}log(age)+\hat{\beta}_{2}female \times log(age) + \hat{\beta}_3 female+\hat{\beta}_4 bachelor+\hat{\beta}_5d 1992 \]

Using this log-log model, a female (female=1) individual has predicted value:
\[\begin{align*} \widehat{log(age)} &= \hat{\beta}_0 + \hat{\beta}_1log(age) + \hat{\beta}_2 \times 1 \times log(age)+ \hat{\beta}_3 \times 1 + \hat{\beta}_4 bachelor+\hat{\beta}_5d 1992\\ & = \hat{\beta}_0 + \left(\hat{\beta}_1 + \hat{\beta}_2 \right)log(age) + \hat{\beta}_3 + \hat{\beta}_4 bachelor+\hat{\beta}_5d 1992 \end{align*}\]

which implies that the elasticity of ahe the with respect to age for females is the sum of the coefficients on log(age).

Computing this partial effect/elasticity based on the estimated coefficients from column (2) of the table in question 7 we obtain an elasticity of: 0.924-0.341=0.583.

Again following the general approach to computing standard errors for nonlinear effects on slides 16-24 of lecture note 8, we can obtain the F-statistic from the following joint test based on the elasticity for females that we just derived:

\(H_0: \beta_1+\beta_2=0 \qquad H_1: \beta_1+\beta_2 \neq0\)

This test yields an F-statistic of 97.439

Ftest=linearHypothesis(reg2,c("log_age+female_log_age=0"),vcov = vcovHC(reg2, "HC1"))

## Recover the Fstat from the joint test results in Ftest
Fstat=Ftest[2,3]
sprintf("Fstat %f", Fstat)
## [1] "Fstat 97.437863"

The standard error for the partial effect/elasticity for females is then computed as:

\(SE\left(\% \Delta\hat{Y}\right)=\dfrac{|\% \Delta \hat{Y}|}{\sqrt{F}} = \dfrac{|0.583|}{\sqrt{97.439}}=0.059\)

Note

we work with percent changes in Y and not just changes in Y in computing the standard error because of the interpretation of the log-log specification; the partial effects are in terms of the percentage change in Y (ahe) associated with a percentage change in X (age).

which implies that the 95% CI for the elasticity of ahe the with respect to age for females is [0.583-1.96 x 0.059 , 0.583-1.96 x 0.059] = [0.467, 0.699].

Combining Interactions: Difference-in-Difference

Recall that in Tutorial 10 we saw a difference-in-differences (DiD) model with Regression 6 of that tutorial which quantified the differential impact of a Bachelor’s degree on earnings between females and males.

Here, we consider another DiD model, this time looking at whether the impact of a Bachelor’s degree on earnings has had a differential impact over time.

Q4

First, construct the following interactive variable

• bachelor_d2012 = bachelor x d2012

Solution

## Create logarthimic variables for ahe and age
mydata1$bachelor_d2012=mydata1$bachelor*mydata1$d2012

Q5

Now run the following regression and interpret the magnitude and statistical significance of the coefficient estimate on bachelor_d2012:

• Regression 3:
• Dependent variable: log_ahe
• Independent variables: bachelor, female, age, d2012, bachelor_d2012

Solution

estimate the model

## DiD regression of log(ahe) 
reg3=lm(log_ahe~bachelor+d2012+bachelor_d2012+female+age,data=mydata1)
cov3=vcovHC(reg3, type = "HC1")    
se3=sqrt(diag(cov3))

Interpreting the results, holding other factors fixed, the statistically-significant 0.385 coefficient estimate on bachelor implies that earnings among individuals with a bachelor’s degree is 38.5% higher on average than those without a bachelor’s degree.

The statistically significant 0.051 coefficient estimate on bachelor_d2012 implies that in 2012 this earnings premium on a bachelor’s education in 2012 incrementally rises by an additional another 5.1%.

Note

Remember, the dependent variable is in logs, so we have a log-linear specification here.



We could have alternatively developed this analysis identically using the d1992 variable instead. With the specification we work with, the difference-in-differences coefficient on bachelor_d2012 has a direct interpretation of the incremental value of a bachelor’s degree on earnings in 2012 relative to its value in 1992.

But we could have obtained an identical analysis working with d1992 and analogously-defined bachelor_d2012 instead.

That is, the bachelor’s earnings premium in 1992 is 38.5% and it grows to 38.5%+5.1%=43.6% in 2012.
In sum, in terms of the “difference-in-differences” model (DID), this quantifies how the earnings difference between bachelor and non-bachelor degree holders (first difference) changes over time between 1992 and 2012 (second difference).
